# Caterpillar Sales Analysis (SQL Project)

## Project Overview
This project analyzes sales data for heavy machinery products to identify key revenue drivers, top customers, and product performance trends.

The analysis was conducted using SQL queries to clean, aggregate, and analyze transactional sales data.

## Objectives
- Identify the most profitable products
- Analyze revenue trends across years
- Determine top-performing customers
- Evaluate regional sales performance

## SQL Techniques Used
- INNER JOIN and CROSS JOIN
- GROUP BY and aggregate functions (SUM, COUNT)
- Window functions (ROW_NUMBER)
- Common Table Expressions (CTEs)
- Subqueries for ranking and filtering results

## Key Insights
- Certain machinery categories contributed the majority of revenue.
- A small group of high-value customers accounted for a significant share of total sales.
- Sales performance varied significantly across regions and product lines.
